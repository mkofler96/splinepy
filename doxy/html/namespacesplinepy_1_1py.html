)

Determine the connectivity from center-vertices, assuming nothing of the underlying grid.

Duplicate Points are not eliminated, assuming that a maximum of two points are equivalent. If this is not the case an exception is thrown. In theory this has complexity O(nlogn) whereas a KDTree has complexity O(n (logn)^dim).

Template Parameters
PhysicalPointTypeType of Point coordinates
ScalarTypeType determining the precision
parametric_dimensiondimension of the object (e.g. surface in 3D)
booleancheck_orientation to check if neighboring elements match structured grid
Parameters
face_center_verticesvertices in the centers of spline-surfaces
parametric_dimensionparametric dimension
metricused for preordering the vertices along a line
tolerancetolerance (distance between two vertices that are joined)
Returns
connectivity as a std::vector<std::array<...>>
